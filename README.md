# Siphiwayinkhosi Mahlalela Portfolio

Cinematic, responsive portfolio for Siphiwayinkhosi Mahlalela, a software
engineer, web developer, and AI automation builder from Eswatini.

## Stack

- React 19
- Vite 6
- Tailwind CSS 4
- GSAP and ScrollTrigger

## Local Development

```bash
npm install
npm run dev
```

The local development server runs at `http://localhost:5173`.

## Checks

```bash
npm run lint
npm run build
```

## Content

Main portfolio content lives in `src/data/portfolio.js`, including contact
details, project case studies, skills, and experience entries.

Static assets live in `public/images` and the downloadable CV is stored in
`public/files`.
